# BizStartAI

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/Starr365/BizStartAI.git
cd BizStartAI
```

### Frontend Setup

To work on the frontend, checkout the frontend branch:

```bash
git checkout `(name of your branch)`
cd bizstart-frontend
npm install
npm run dev
```

The frontend will be available at `http://localhost:5173` (or the port specified in your configuration).
