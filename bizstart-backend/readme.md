# BizStartAI Backend

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/Starr365/BizStartAI.git
cd BizStartAI
```

### Backend Setup

To work on the backend, checkout the backend branch:

```bash
git checkout `(name of your branch)`
cd bizstart-backend
npm install
npm start
```

The backend API will be available at `http://localhost:5000` (or the port specified in your configuration).

## Development

The project backend code is maintained in the `backend` folder.
